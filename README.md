# Periodontology — Landing Page

Single-file landing page for the **Active Recall Dentistry — Periodontology** study system
(Dr. Manuela Rapani), used to validate interest via a waitlist.

- `index.html` — self-contained landing page (images embedded), with an email waitlist form.
- Waitlist submissions are collected via [FormSubmit](https://formsubmit.co/) and delivered to the author's email.

Published with GitHub Pages.
